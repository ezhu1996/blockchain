Application Description

Blockchain is heavily implemented in cryptocurrency applications and settings. Therefore, we implement an app that will allow cryptocurrency users and holders to be able to easily store all their bitcoin addresses and have a real time view of how those bitcoin values translate to other popular cryptocurrencies and USD. On opening of the app, a user will view a list of all notable cryptocurrencies and their translated value to their appropriate USD equivalence. A user may also translate a specified amount of bitcoin to other paper currencies using the swap page view. As a default, we have set the swap page to translate 1 BTC.

Furthermore, to invoke more of a blockchain approach, a user can also create an account and login to the app in order to store several bitcoin addresses. The app will then read each address and return the value of bitcoin for said address. Then, the user may return to the swap screen and will notice that the bitcoin value is now translated to all other present cryptocurrencies and USD.

The significance of such an app can be attributed to individuals trading cryptocurrency. Specifically, users may want to trade certain cryptocurrencies for others depending on their current market value. Our app allows such individuals to have immediate knowledge of the real-time cryptocurrency market to make an appropriate decision.
